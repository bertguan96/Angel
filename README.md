# 一款基于PHP的快速开发web框架

-------------------

[PHP | 敏捷开发 | 框架] 

![](https://img.shields.io/appveyor/build/guanjiangtao/Angel)

####  主要模块

- **数据库层次** ：数据库封装了常用SQL语句在，封装了查询器（用于执行字写书写的SQL语句），XML解析器（用于解析XML文件中的SQL语句，再注入进入方法）；
- **常用工具层次** ：提供字符串转换工具，时间转换工具，通用XML解析工具，待新增更多工具
- **热门依赖整合** ：整合PHP热门依赖。
- **前端** ：采用主流前端框架vue设计搭建，整合通用组件。
